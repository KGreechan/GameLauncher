# Game Launcher / Touhou Randomiser

### Touhou Randomiser

A randomiser for choosing runs in touhou games for when i get indecisive.

- Only works for games i own:
  - Touhou 6
  - Touhou 7
  - Touhou 16
  
- Generates a random game, character to use and a shot type to use
